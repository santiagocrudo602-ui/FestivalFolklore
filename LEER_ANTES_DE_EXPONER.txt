=== INSTRUCCIONES PARA LA EXPOSICIÓN DEL PROYECTO ===

Para que el prototipo funcione correctamente en la computadora donde vayas a exponer (si es distinta a la tuya), debes seguir estos 3 pasos:

1. BASE DE DATOS (MySQL)
-------------------------------------------------
La computadora de la exposición debe tener instalado un gestor de base de datos (como XAMPP, WAMP o MySQL Server).
- Abre tu gestor de base de datos (ej. phpMyAdmin).
- Importa el archivo que está en la carpeta "database/festival_db.sql".
- Esto creará la base de datos "festival_db" junto con todas las tablas y datos de prueba.

2. LEVANTAR EL SERVIDOR (Node.js)
-------------------------------------------------
La computadora debe tener instalado Node.js.
- Abre una terminal (Símbolo del sistema o PowerShell).
- Navega hasta esta carpeta (la carpeta "folklore" o "Prototipo_Folklore").
- Si copiaste la carpeta "node_modules" en el pendrive, simplemente ejecuta:
  node server.js
- Si NO copiaste "node_modules" (pesa un poco), primero debes instalar las dependencias ejecutando:
  npm install
  Y luego iniciar el servidor con:
  node server.js

3. MOSTRAR EL SISTEMA
-------------------------------------------------
- Una vez que la terminal diga "Servidor iniciado: http://localhost:3000", abre tu navegador.
- Ve a tu carpeta "Dashboard" y abre el archivo "index.html" haciéndole doble clic.
- Muestra el Dashboard ejecutivo y, cuando sea el momento, haz clic en la pestaña "🚀 Ver Prototipos".
- El botón te redirigirá automáticamente al sistema en localhost:3000.

¡Éxitos en la presentación!
